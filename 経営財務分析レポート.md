# 第1章: 売上分解分析

## 1.1 売上構造の基本式

```
売上高 = 客数 × 客単価
       = (1日平均客数 × 営業日数) × 客単価
```

## 1.2 月次売上分解（推定）

### 前提条件
- 営業日数: 30日（11月）、30日（12月）、30日（1月）
- 客単価（仮定）: ¥2,750
- 営業時間: 14:00-24:00（10時間）

### 分解結果

| 月 | 売上 | 推定客数 | 1日平均客数 | 客単価（仮定） | 前月比 |
|:---|---:|---:|---:|---:|:---:|
| 11月 | 610,000円 | 222人 | **7.4人/日** | 2,750円 | - |
| 12月 | 550,000円 | 200人 | **6.7人/日** | 2,750円 | -9.8% |
| 1月 | 400,000円 | 145人 | **4.8人/日** | 2,750円 | -27.3% |

| 期間 | 客数減少 | 1日平均減少 | 売上減少額 |

### 重要な発見
1. **客数減少が加速している**（12月: -0.7人/日 → 1月: -1.8人/日）
2. 1月の減少幅は12月の**2.6倍**に拡大

## 1.3 客単価構造分析

### 理論的客単価パターン

#### パターンA: ミニマム消費
```
シーシャ基本（2名以上）  ¥2,400
スタンダードドリンク    ¥500
SNS投稿割引          -¥200
─────────────────────
合計                ¥2,700
```

#### パターンB: 標準消費
```
シーシャ基本          ¥2,700
スタンダードドリンク    ¥500
（SNS投稿なし）
─────────────────────
合計                ¥3,200
```

#### パターンC: 追加オプション型
```
シーシャ基本          ¥2,700
ダークリーフ          ¥300
スタンダードドリンク    ¥500
追加ドリンク          ¥500
─────────────────────
合計                ¥4,000
```

#### パターンD: プレミアム型
```
シーシャ基本          ¥2,700
オールダーク          ¥1,000
スペシャルドリンク     ¥700
追加ドリンク          ¥700
─────────────────────
合計                ¥5,100
```

#### パターンE: トップ替え型
```
シーシャ基本          ¥2,700
トップ替え            ¥2,000
スタンダードドリンク    ¥500
追加ドリンク          ¥500
─────────────────────
合計                ¥5,700
```

### 客単価¥2,750（推定）の意味

現在の客単価¥2,750は、**パターンAとパターンBの中間**に位置しています。

**推論**:
- 大半の顧客がSNS投稿割引を利用している可能性が高い
- 追加注文（ドリンク・オプション）の発生率が低い
- トップ替えは稀

### 客単価向上のボトルネック分析

| ボトルネック | 推定影響 | 改善インパクト |
|:---|:---|:---|
| **SNS投稿割引の多用** | -¥200/客 | 割引なし比率を30%→50%にすると+¥40/客 |
| **追加ドリンク発生率の低さ** | 機会損失¥500/客 | 発生率を10%→30%にすると+¥100/客 |
| **オプション訴求不足** | 機会損失¥300-1,000/客 | ダークリーフ発生率を5%→15%にすると+¥30/客 |

**合計改善余地: 客単価 ¥2,750 → ¥2,920（+¥170、+6.2%）**

# 第3章: 季節要因 vs 構造要因の分離

## 3.1 売上減少の要因分解（仮説）

### 減少パターンの観察

| 月 | 売上 | 前月比 | 減少率 | 減少額 |
|:---|---:|:---:|---:|---:|
| 11月→12月 | 61万→55万 | -9.8% | 小幅減少 | -6万円 |
| 12月→1月 | 55万→40万 | -27.3% | **急減** | -15万円 |

### パターン解釈

**減少パターンは2段階**:
1. 11月→12月: 緩やかな減少（-9.8%）
2. 12月→1月: 急激な減少（-27.3%）

このパターンから推測される要因:

#### 要因A: 季節要因（ニッパチ閑散期）
- 1月は飲食業界で最も厳しい「ニッパチ」の月
- 年末年始の出費後の財布の紐締め
- 外出控え（寒さ、正月疲れ）
- **影響推定: -15〜20%**

#### 要因B: 既存顧客の離脱（構造的問題）
- リピート促進施策の不在（LINE等なし）
- 顧客との接点維持の仕組みがない
- 一度離れた顧客が戻るきっかけがない
- **影響推定: -10〜15%**

#### 要因C: 集客チャネルの弱体化
- SNS投稿の減少可能性
- 口コミの自然減
- Googleマップ順位の変動可能性
- **影響推定: -5〜10%**

### 要因別影響度（推定）

| 要因 | 影響度 | 回復可能性 | 対策優先度 |
|:---|:---:|:---|:---:|
| **季節要因** | -20% | 2-3月で自然回復 | P2（待つ） |
| **既存顧客離脱** | -10% | 施策で即時回復可 | **P0（緊急）** |
| **集客弱体化** | -5% | 1-2ヶ月で回復 | P1（重要） |

### 分離の試算

#### 11月を基準（100%）とした場合
```
11月: 61万円（100%）

12月: 55万円（90.2%）
  ├ 季節要因: -3%
  ├ 既存顧客離脱: -4%
  └ 集客弱体化: -3%

1月: 40万円（65.6%）
  ├ 季節要因: -20%（ニッパチ本格化）
  ├ 既存顧客離脱: -10%（累積）
  └ 集客弱体化: -5%（累積）
```

**重要な示唆**:
- 季節要因の影響は-20%程度と推定
- 残りの-14.4%は**構造的問題**
- **何もしなければ2月も同様の減少が続く**
