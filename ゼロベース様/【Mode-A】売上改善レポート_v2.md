# 第1章: 現状分析

## 1.1 売上推移

| 月 | 売上 | 前月比 |
|:---|---:|:---:|
| 11月 | 61万円 | - |
| 12月 | 55万円 | -9.8% |
| 1月 | 40万円 | -27.3% |

3ヶ月で21万円の売上減少が発生しています。特に1月の落ち込み（-27%）が顕著です。

## 1.2 客数の推定

客単価を2,750円と仮定した場合の推定値です。

| 月 | 推定客数 | 1日平均 |
|:---|---:|---:|
| 11月 | 222人 | 7.4人 |
| 12月 | 200人 | 6.7人 |
| 1月 | 145人 | 4.8人 |

1日あたり約2.6人の客数減少が発生しています。

## 1.3 ワンオペの構造的上限

15席・ワンオペ・10時間営業の条件下での上限を試算しました。

| 指標 | 数値 |
|:---|---:|
| 同時対応可能人数 | 5〜7名 |
| 現実的な回転数 | 2〜3回転/日 |
| **月商上限** | **70〜80万円** |

現状40万円は上限の約50%であり、**ワンオペのまま70万円までは回復余地があります**。100万円以上を目指す場合はスタッフ増員が必要です。

# 第2章: 売上減少の原因分析

## 2.1 主因の特定

| 優先度 | 主因 | 根拠 | 確信度 |
|:---:|:---|:---|:---:|
| **1** | **季節要因** | 1〜2月は「ニッパチ」と呼ばれる閑散期。1月の急落（-27%）と一致 | 高 |
| **2** | **既存顧客の離脱** | リピート促進施策（LINE等）がなく、一度離れた顧客が戻らない | 中 |
| **3** | **集客チャネルの弱体化** | 冬場の外出控えにより、全チャネルが同時に弱体化 | 中 |

## 2.2 分析の区分

### 検証的分析（事実として確認）

- 売上は3ヶ月連続で減少している（事実）
- 減少率は加速している：11→12月は-10%、12→1月は-27%（事実）
- ワンオペ上限は70〜80万円（業界データに基づく推定）

### 探索的分析（仮説として提示）

- 季節要因が売上減少の10〜15%を説明する可能性がある（仮説）
- 既存顧客の自然離脱が進行している可能性がある（仮説）
- 前年同月比での検証ができればより正確な診断が可能（検証方法）

## 3.1 72時間以内の止血施策

### 施策1: 常連10名に直接連絡【最優先】

| 項目 | 内容 |
|:---|:---|
| 目的 | 離れかけている既存顧客を即座に呼び戻す |
| 実施内容 | スマホの連絡先・LINE履歴から常連を抽出し、個別に連絡 |
| 工数 | 1時間 |
| 期待効果 | 1〜3名の即時再来店（売上+8,000〜25,000円） |

**連絡例文**:
> 「〇〇さん、お久しぶりです！最近寒いですね。今月から冬限定のホットドリンク始めたので、良かったらまた来てください」

### 施策2: 追加ドリンク声かけの習慣化

| 項目 | 内容 |
|:---|:---|
| 目的 | 客単価を即座に向上させる |
| 実施内容 | シーシャ提供時に「ドリンクいかがですか？」を必ず言う |
| 工数 | 0分（習慣化のみ） |
| 期待効果 | 客単価+300〜500円 |

**声かけ例**:
> 「シーシャお持ちしますね。今日寒いので、ホットドリンクもいかがですか？チャイが人気ですよ」

### 施策3: 冬季限定ホットドリンク追加

| 項目 | 内容 |
|:---|:---|
| 目的 | 寒い時期の来店理由を作る |
| 実施内容 | チャイ、ホットワイン、ホットカクテル等を3種追加 |
| 工数 | 仕入れ1時間 |
| 期待効果 | 客単価向上+「温まれる店」という訴求 |

# 第4章: 今やらないこと

以下は、現段階では実施しないことを推奨します。

| やらないこと | 理由 |
|:---|:---|
| **大規模な広告投資** | 原価・費用が不明な状態で追加支出は危険。まず既存顧客の活性化を優先 |
| **値下げ・割引強化** | 客単価向上が優先。安易な値下げは利益を圧迫する |
| **スタッフ採用** | 月商70万円到達後に検討。現段階では人件費増のリスクが高い |
| **メニューの大幅変更** | 現状把握が不十分。まずデータを取ってから判断 |
| **営業時間の変更** | ワンオペの負荷を考慮し、現状維持。データで判断 |
| **新規事業・大型イベント** | まず本業の立て直しに集中。体力がついてから検討 |

# 第6章: 本レポートの限界

## 6.1 データ制約による限界

| 不足データ | 影響 |
|:---|:---|
| **原価・粗利** | 「赤字」かどうか判断不能。本分析は売上回復に限定 |
| **固定費（家賃等）** | 損益分岐点が算出不能 |
| **日次データ** | 曜日・時間帯の傾向分析不能 |
| **客単価実績** | 推定値（2,750円）に依存 |
| **前年同月データ** | 季節要因の正確な影響度が算出不能 |

## 6.2 本レポートで断定できないこと

- 店舗が「赤字」かどうか
- 資金繰りに問題があるかどうか
- 季節要因と構造要因の正確な比率
- 競合の影響度

## 6.3 追加データ取得の推奨

より精度の高い分析のため、以下のデータ取得を推奨します。

| 優先度 | データ | 取得方法 |
|:---:|:---|:---|
| 1 | 月次PL（直近3ヶ月） | 会計ソフト or 手書き帳簿 |
| 2 | 家賃・固定費一覧 | 契約書 or 通帳確認 |
| 3 | 日次売上（直近1ヶ月） | レジ記録 or 手書きメモ |
| 4 | 前年同月売上 | 過去帳簿 |
